# Personal Profile Website

This project is a simple personal profile website created using HTML and CSS.  
It shows basic information about me, my skills, projects, and contact details.

## Files in this Project

- **index.html** – The main webpage structure.
- **style.css** – The stylesheet used to design the webpage.
- **README.md** – Information about the project.

## Features

- Personal introduction
- Skills section
- Projects section
- Contact information
- Simple and clean design

## Technologies Used

- HTML
- CSS

## Purpose of the Project

This project was created as a beginner web development practice and to build a simple personal portfolio page that can be hosted on GitHub.
